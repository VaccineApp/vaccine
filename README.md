Vaccine
=======

![hero](res/hero.png)

This is an imageboard browser for Linux that is written in Vala and uses GTK.
Please contribute and report bugs.

Libraries
---

* GTK+ 3
* json-glib
* libsoup
* gee (no, not the board)
* gstreamer-1.0
* gtkgst-1.0

Try it
---
```Bash
$ ./autogen.sh
$ make -j`nproc` run
```
